# 东莞理工学院疫情打卡

## Usage
- Fork本项目  
- Actions中启用workflow  
- 打开Settings>>Secrect>>Actions>>New repository secret
```
(Name)     （Secret）
//添加以下几个
USERNAME    (学号/工号)
PASSWORD    (密码)
```
